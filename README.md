# Vite Boilerplate

This is a simple boilerplate that aims to provide a setup for a large project using **Vite**

This boilerplate includes:

- TailwindCSS
- Typescript with lints and prettier
- i18n
- MaterialUI
- Routing
- A basic JenkinsFile and and pipeline config

Hooks:

- useForm
- useAuth

Components:

- Input
- Password
